使用autopep8.py对代码进行pep8风格格式化
------------------------------------------------------------------------------

1. 启用autopep8.py: preferences > window > preferences > pydev > editor > code style > code formatter > use autopep8.py
2. 使用快捷键 Ctrl + Shift + F 对当前Python脚本进行格式化。或者手动使用菜单中的 souce > format code。
