Install
------------------------------------------------------------------------------

``dev_exp_share`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install dev_exp_share

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade dev_exp_share
