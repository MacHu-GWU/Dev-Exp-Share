Pytest Mark
==============================================================================

Ref:

- https://docs.pytest.org/en/latest/how-to/mark.html


What is Mark
------------------------------------------------------------------------------
Mark 是 pytest 中为你的测试用例提供 metadata 的机制. pytest 的 fixture 本质上就是 mark 的一种. 当然内置的 skip, xfail, parameterize 也非常有用.
