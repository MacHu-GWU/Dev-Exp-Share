What is Redshift
------------------------------------------------------------------------------

- 使用超大规模并行, 为 GB / TB / PB / EB 级数据 提供 **亚秒级 (1-10s)** 查询响应速度.
- 使用标准 SQL 查询语言
- 使用列存储
