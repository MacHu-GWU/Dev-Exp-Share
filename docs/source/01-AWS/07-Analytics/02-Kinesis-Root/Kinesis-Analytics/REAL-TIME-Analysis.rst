How many user signed up so far?

::

    SELECT COUNT(*)
    FROM events
    WHERE
        events.event_name = "sign_up"

How many new user signed up this month so far?

::

    SELECT COUNT(*)
    FROM events
    WHERE
        events.event_time >= DATETIME(TODAY().month, 1, 1)
        AND events.event_name = "sign_up"
