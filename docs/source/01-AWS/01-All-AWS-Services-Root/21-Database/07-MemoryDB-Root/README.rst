What is AWS MemoryDB
------------------------------------------------------------------------------

AWS MemoryDB 是 AWS 研发的类 Redis, 跟 Redis 语法和数据结构兼容, 并且云原生的内存数据库.



Memory / Redis 的应用场景:

- cache for hot data
- time to live
- counter
- top N ranking
- distributive lock
