观察者模式 和 订阅模式 (Observer vs Publisher Subscriber Pattern) 的区别
==============================================================================

一句话解释:

- Observer 没有 broker, observer 和 observable 松耦合, 异步通信.
- Publisher Subscribe 有 broker, 都和 broker 松耦合.
